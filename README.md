# RedDeAmistades-Grafos

Indicaciones:

Desarrollar un programa que simule una red de amigos utilizando un grafo no dirigido. 

1-. Permitir al usuario ingresar nombres (nodos) y relaciones (aristas).

2-. Mostrar las conexiones de cada persona.

3-. Validar si un nodo existe antes de conectarlo.

4-. Visualizar el recorrido completo del grafo.

5-. Buscar si dos personas están conectadas (directa o indirectamente).
Usar BFS o DFS para comprobar si existe un camino entre dos personas. 

6-. Mostrar el grado de cada persona (número de amistades). 
El grado de un nodo es el número de conexiones directas que tiene. 

7-. Encontrar el camino más corto entre dos personas. 
Usar BFS para devolver el camino exacto (lista de nombres). 

8-. Detectar grupos o comunidades (componentes conexos). 
Cada grupo representa personas conectadas directa o indirectamente entre sí.
